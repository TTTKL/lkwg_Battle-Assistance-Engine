## 🔧 技术实现亮点

### 数据结构设计
```javascript
// 精灵战斗数据 - 扩展字段
{
  marks: {},        // 印记系统
  status: null,     // 状态异常
  statusTurns: 0,   // 状态持续
}

// 全局战斗数据 - 扩展字段
{
  weather: null,    // 天气系统
  weatherTurns: 0,  // 天气持续
}
```

### 伤害计算公式
```
最终伤害 = 攻击/防御 × 0.9 × 
  (基础威力 × 应对倍率 + 威力加成 + 印记加成) × 
  能力等级 × 威力buff × 本系加成 × 克制关系 × 
  天气影响 × 特性倍率 × 连击数 × (1 - 减伤)
```

### 核心函数
- `triggerTraits()` - 特性触发处理
- `applyEff()` - 效果应用 (支持印记/状态/天气)
- `execAction()` - 行动执行 (状态检查)
- 伤害计算 - 集成所有加成系统
